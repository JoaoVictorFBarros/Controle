# Clone o repositório
```bash
git clone https://github.com/JoaoVictorFBarros/Controle.git
```
# Instale as dependências
```bash
npm install
```
# Rodando a aplicação
```bash
npm start
```
para rodar a versão de desenvolvimento. alternativamente pode-se criar a build e rodar a partir dela com:
```bash
npm run build
```
```bash
serve -s build
```
